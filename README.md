# Tasker web

Task_1:

Given two arrays of strings 1 and 2 return a  new sorted array in lexicographical order of the strings of 1 which are substrings of strings of 2.

Example:

arp, live, strong

lively, alive, harp, sharp, armstrong

returns:

arp, live, strong
